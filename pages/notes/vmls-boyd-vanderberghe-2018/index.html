Part I : Vectors

1. Vectors

  • A vector is an ordered, finite list of numbers.

    • In coding, it is usually represented by arrays or lists (and not tuples).

  • Vector notations can wildly differ among authors, so caution is advised.

  • Some notations don’t distinguish between a 1-vector and scalars (numbers). In coding, they’re in different data structures.

  • There’s vectors like zero, unit, one, sparse vectors.

  • Adding/subtracting vectors is different from stacking their contents.

  • The inner product (or dot product) of n-vectors xxx and yyy is written as xTyx^TyxTy

  • Sparse vectors reduce computation time; both Python and Julia have them as dedicated data structures.

2. Linear Functions

  • Functions that take in vectors are simply written as f(x)f(x)f(x) with xxx as an n-vector.

  • Example: inner product function f(x)=aTxf(x) = a^Txf(x)=aTx

    • Can be seen as a weighted sum of elements of xxx

  • A linear function f(x)f(x)f(x) is when:

    • f(ax)=af(x)f(ax) = af(x)f(ax)=af(x) for scalar aaa and n-vector xxx

    • f(x+y)=f(x)+f(y)f(x+y) = f(x) + f(y)f(x+y)=f(x)+f(y) for n-vectors xxx and yyy

  • Example: mean() is a linear function, max() is not, median() is not.

  • A differentiable function (doesn't need to be linear) can be approached by a linear function with Taylor approximation.

  • Regression can be written with vectors as f(x)=xTβ+vf(x) = x^T\beta + vf(x)=xTβ+v (an affine function)

4. Clustering

  • Clustering outside of simple cases:

    • In almost all applications nnn is larger than 2 (we can’t just scatterplot the values).

    • There will be some or even many points between clusters.

    • In real examples the best number of kkk clusters will be less clear.

  • Formalizing a clustering objective:

    • All vectors xix_ixi​ get clustered into clusters ccc , where cic_{i}ci​ is the group where xix_ixi​ is assigned to.

    • With each cluster ccc there is a group representative zciz_{c_{i}}zci​​ (doesn’t need to be one of xix_ixi​) which is close to each vectors xix_{i}xi​ in that group (small distance ∣∣xi−zci∣∣||x_{i} - z_{c_{i}}||∣∣xi​−zci​​∣∣ ).
