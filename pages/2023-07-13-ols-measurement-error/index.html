How Measurement Error Affects OLS

this is another data science question I saw on twitter:

basically we're trying to answer how α\alphaα, β\betaβ, etc changes if errors are added to the yyy values.

note that ryxcommar's notation for normal distribution is N(μ,σ)N(\mu,\sigma)N(μ,σ) instead of N(μ,σ2)N(\mu,\sigma^2)N(μ,σ2). we'll be using this for the answer.

so y∗=α+βx+ϵ+uy^* = \alpha + \beta x + \epsilon + uy∗=α+βx+ϵ+u where uuu is measurement error distributed N(2,2)N(2,2)N(2,2)

since xxx isn't affected by the measurement error, β\betaβ is also unaffected.

which means uuu only affects α\alphaα & ϵ\epsilonϵ.

OLS residuals always have mean 000, because the constant is carried over to the α\alphaα term. so α∗=α+2\alpha^* = \alpha + 2α∗=α+2. and ϵ∗∼N(0,1)+N(0,2)∼N(0,5)+cov(e,u)\epsilon^* \sim N(0,1) + N(0,2) \sim N(0,\sqrt{5}) + cov(e,u)ϵ∗∼N(0,1)+N(0,2)∼N(0,5​)+cov(e,u). this is why the second correction is nice to have, so ϵ∼N(0,5)\epsilon \sim N(0,\sqrt{5})ϵ∼N(0,5​)

in conclusion:

  • α∗=α+2\alpha^* = \alpha + 2α∗=α+2

  • β∗=β\beta^* = \betaβ∗=β

  • μ∗=μ=0\mu^* = \mu = 0μ∗=μ=0

  • σ∗=σ+22\sigma^* = \sqrt{\sigma + 2^2}σ∗=σ+22​
